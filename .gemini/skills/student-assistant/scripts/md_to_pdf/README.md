# Markdown to PDF Converter

A simple CLI tool to convert Markdown files to PDF using Node.js.

## Prerequisites

- Node.js installed on your system.

## Setup

1.  Navigate to this directory:
    ```bash
    cd md_to_pdf
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Usage

You can run the script directly as an executable:

```bash
./convert.js <path-to-markdown-file>
```

Or using `node`:

```bash
node convert.js <path-to-markdown-file>
```

### Example

```bash
./convert.js ../README.md
```

This will create a `../README.pdf` file.
